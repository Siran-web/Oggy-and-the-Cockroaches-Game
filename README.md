# 🕹️ Oggy and the Cockroaches Game

A fun browser-based game built with **React + Vite** inspired by the cartoon *Oggy and the Cockroaches*.  
Your goal is to help **Oggy** chase away the mischievous cockroaches and score as many points as possible!

---

## 🚀 Features
- 🎨 Cartoon-themed graphics with Oggy and Cockroaches.
- 🎶 Background music and sound effects.
- 🖱️ Simple click/tap gameplay.
- ⚡ Built using **React**, **Vite**, and **Tailwind CSS**.
- 🔄 Fast reload with Vite HMR (Hot Module Replacement).

---

## 🛠️ Tech Stack
- **React** – UI components
- **Vite** – Development & bundling
- **Tailwind CSS** – Styling
- **JavaScript (ES6+)**
- **HTML5 / CSS3**

## 🔧 Setup Instructions

### 1. Clone the repository
```
git clone https://github.com/Siran-web/Oggy-and-the-Cockroaches-Game.git
cd Oggy-and-the-Cockroaches-Game/my-project.
```

### 2. Install dependencies
```npm install```

### 3. Run in development mode
```npm run dev```
Now open the local server (default: http://localhost:5173
).
